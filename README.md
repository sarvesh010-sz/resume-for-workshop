# Sarvesh's Resume

This is a simple HTML and CSS resume template for Sarvesh.

## Usage

Open `index.html` in your web browser to view the resume.

## Files

- `index.html`: The main HTML structure of the resume.
- `styles.css`: The CSS styles for the resume.

## Troubleshooting

- Ensure both `index.html` and `styles.css` are in the same directory.
- If styles don't load, check the link in the HTML file.
